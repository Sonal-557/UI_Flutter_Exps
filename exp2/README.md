# Experiment 2: Widgets & Layouts

### Aim
a) Explore widgets (Text, Image, Container, Icon)  
b) Implement Row, Column, Stack layouts.

### Description
a) Basic screen with Text, network Image, Container. 
b) Row of icons and a Stack with positioned containers.

### Files
- `lib/main.dart` (experiment UI)

### Steps / Commands
- Create project: `flutter create exp2`  
- Put code in `lib/main.dart` (we used Image.network)  
- Run on web: 
flutter pub get
flutter run -d chrome

### Output
- App shows "Hello, Flutter!", Flutter logo (network), styled Container, Row icons, Stack elements.
