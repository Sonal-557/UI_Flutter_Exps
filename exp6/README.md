# Experiment 6: Custom Widgets & Themes

### Aim
a) Create reusable custom widgets  
b) Apply app-wide theme and custom styles

### Description
a) `main_a.dart`: `ColoredCard` custom widget used in a list.  
b) `main_b.dart`: `ThemeData` with `elevatedButtonTheme` and text style; widgets inherit theme.

### Files
- `lib/main_a.dart` (6a)  
- `lib/main_b.dart` (6b)

### Steps / Commands
flutter pub get
flutter run -t lib/main_a.dart -d chrome
flutter run -t lib/main_b.dart -d chrome

### Output
- Reusable cards displayed; theme styling applied to button/text.
