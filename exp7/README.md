# Experiment 7: Forms & Validation

### Aim
a) Design a form with input fields  
b) Implement form validation and error handling

### Description
a) `main_a.dart`: Simple form UI with TextField (Name, Email, Password).  
b) `main_b.dart`: Uses `Form`, `TextFormField`, validators, and shows SnackBar on successful validation.

### Files
- `lib/main_a.dart` (7a)  
- `lib/main_b.dart` (7b)

### Steps / Commands
flutter pub get
flutter run -t lib/main_a.dart -d chrome
flutter run -t lib/main_b.dart -d chrome

### Output
- 7a: Form UI shown.  
- 7b: Validation messages for empty/invalid fields and success SnackBar on submit.
