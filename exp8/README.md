# Experiment 8: Animations

### Aim
a) Add animations using Flutter animation framework  
b) Experiment with fade, slide, etc.

### Description
a) `main_a.dart`: Size animation using `AnimationController` and `Tween`.  
b) `main_b.dart`: `SlideTransition` + `FadeTransition` combined.

### Files
- `lib/main_a.dart` (8a)  
- `lib/main_b.dart` (8b)

### Steps / Commands
flutter pub get
flutter run -t lib/main_a.dart -d chrome
flutter run -t lib/main_b.dart -d chrome

### Output
- Animated container resizing (8a).  
- Combined slide+fade animation (8b).
