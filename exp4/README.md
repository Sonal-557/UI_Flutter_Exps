# Experiment 4: Navigation

### Aim
a) Setup navigation between screens with Navigator.push/pop  
b) Implement navigation using named routes

### Description
a) `main_a.dart`: Home screen -> Second screen using `Navigator.push` and `Navigator.pop`.  
b) `main_b.dart`: Uses `routes` map and `Navigator.pushNamed`.

### Files
- `lib/main_a.dart` (4a)  
- `lib/main_b.dart` (4b)

### Steps / Commands
- Create `exp4`, add files, then:
flutter pub get
flutter run -t lib/main_a.dart -d chrome # for 4a
flutter run -t lib/main_b.dart -d chrome # for 4b

### Output
- Buttons navigate between screens; named routes work via route names.
